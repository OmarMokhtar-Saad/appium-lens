# Support

Thanks for using **Appium Lens**! Here's how to get help.

## Reporting bugs & requesting features

Please [open an issue](../../issues/new/choose) using one of the templates:

- 🐛 **Bug Report** — something isn't working as expected
- 💡 **Feature Request** — an idea or improvement

Before filing, search [existing issues](../../issues) to avoid duplicates.

## Direct contact

For private, security, or licensing questions, email **support@appiumlens.com**.

## Helpful info to include

- Plugin version (see **Settings → Plugins → Appium Lens**)
- IDE name and version (**Help → About**)
- Operating system
- Relevant logs (**Help → Show Log in Finder/Explorer**) or plugin terminal output
